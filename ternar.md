# Тернарные операторы:

| Задача                        | Оператор | Пример                               |
|------------------------------|----------|----------------------------------------|
| Простое условие              | `? :`    | `age >= 18 ? 'yes' : 'no'`            |
| Условный рендеринг           | `&&`     | `{isOpen && <Modal />}`               |
| Значение по умолчанию (строго) | `??`     | `count ?? 0`                          |
| Значение по умолчанию (старый) | `||`     | `name || 'Guest'`                     |
| Безопасный доступ            | `?.`     | `user?.address?.city`                 |
| Сложная логика               | `if-else`| Не тернарник!                         |
